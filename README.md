# 💰 Personal Finance Tracker Dashboard

A responsive and interactive web application designed to help users efficiently manage personal finances by tracking income and expenses in real time. The application provides a clean dashboard experience with visual insights, persistent storage, and an intuitive user interface.

---

## 🚀 Project Overview

The Personal Finance Tracker Dashboard allows users to record financial transactions, categorize them as income or expense, and instantly view updated balances. It enhances financial awareness by visually representing data using charts and ensures data persistence using browser storage, making it a practical and user-friendly personal budgeting tool.

---

## ✨ Key Features

- ➕ Add income and expense transactions with description and amount  
- ✏️ Edit existing transactions for quick updates  
- ❌ Delete unwanted transactions instantly  
- 📊 Real-time calculation of total balance, income, and expenses  
- 📈 Interactive doughnut chart for financial overview  
- 🌙 Dark/Light mode toggle for better user experience  
- 💾 Data persistence using local storage (no data loss on refresh)  
- 📱 Fully responsive design for mobile and desktop users  

---

## 🛠️ Tech Stack

- HTML – Structure of the application  
- CSS – Styling with modern UI and responsive design  
- JavaScript – Core logic, DOM manipulation, and interactivity  
- Chart.js – Data visualization (income vs expense chart)  

---

## 🎯 Purpose of the Project

This project was built to strengthen core front-end development skills including JavaScript logic building, DOM manipulation, event handling, and working with browser storage. It also focuses on creating a real-world application experience with a clean and professional UI.

---

## 📂 How It Works

1. User enters transaction details (income or expense)  
2. Data is processed and stored in an array and local storage  
3. UI updates dynamically without page reload  
4. Financial summary and chart update in real time  
5. Data remains saved even after refreshing the browser
